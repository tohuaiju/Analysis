### 对于网站登录
由于项目式部署在pythonanywhere上面，免费用户的mysql是不能用于外部连机的，当时为了方便，账号信息直接放在了static下面的一个json里面，信息有暴漏风险，不要用自己真是邮箱密码去注册，随便邮箱格式就可以。实训结束，改接着着手考研了，没时间修改了，见谅。
